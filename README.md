# HE CARES Foundation – Empowerment & Protection Platform

---

## 🚧 Project Status: In Progress

> **Note:**  
> The backend (FastAPI) is **currently under active development**.  
> Many dynamic features (forms, authentication, live data) are not yet functional.  
> **Frontend (Angular)** sections and UI are being scaffolded and integrated as backend endpoints are built.

---

## 🌍 Overview

**HE CARES Foundation** is a modern web platform dedicated to empowering and protecting women and children in South Africa, uniquely mobilizing men as visible advocates for change.  
The system will provide crisis support, resources, education, and a vibrant community hub for advocacy and action.

---

## ✨ Key Features (Planned & In Progress)

- Responsive Angular frontend, accessible on all devices
- FastAPI backend (under development)
- Volunteer signup and management
- Donation and fundraising tools
- Crisis support & referrals
- Educational empowerment programs
- Advocacy campaigns & community involvement
- Impact stories, testimonials, and analytics
- Secure contact and feedback forms

## 🛠️ Tech Stack

### Frontend

- **Angular:** Modern SPA framework
- **SCSS:** Modular, maintainable styles
- **Responsive Design:** Optimized for all devices

### Backend

- **FastAPI (Python):** (In progress)
- **Planned:** Supabase/PostgreSQL for data, SMTP for email

## 🚀 Getting Started

> **Note:** Project is not yet ready for public/local deployment.

### To preview the UI:

1. Clone the repository:
    ```bash
    git clone git@github.com:PixieStack/he-cares-foundation.git
    cd he-cares-foundation
    ```
2. Run the frontend:
    ```bash
    cd frontend
    npm install
    ng serve --open
    ```
